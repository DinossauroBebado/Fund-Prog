### Atalhos uteis para o Codeblocks 
Para comentar um bloco de codigo, seleciona o bloco e <b> Ctrl+Shift + C </b>
Analogamente para descomentar <b>Ctrl + Shift + X </b> 
Atalho para rodar o codigo <b> F9 </b>
